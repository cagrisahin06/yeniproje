Response Levels

Log only, no user impact

Level 2 - Warn

User notification, continue

Level 3 - Block

Block action, admin alert

Level 4 - Isolate

Device isolation, SOC alert

Decision Matrix

func DetermineResponse(threat) {
    score = CalculateRiskScore(threat)
    
    if (score < 30) {
        return MONITOR
    } else if (score < 60) {
        return WARN
    } else if (score < 80) {
        return BLOCK
    } else {
        return ISOLATE
    }
}
